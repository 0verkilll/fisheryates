# fisheryates

Pure-Go Fisher–Yates shuffle, derived from the Java F5 reference with an unbiased index-reduction fix. It produces the password-driven coefficient permutation that F5 uses.

## Install

```bash
go get github.com/0verkilll/fisheryates
```
